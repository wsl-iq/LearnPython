المتغيرات في Python

السابق: الطباعة والإدخال | التالي: أنواع البيانات | الرئيسية

ما هي المتغيرات؟

المتغير هو اسم يُستخدم لتخزين قيمة معينة في الذاكرة، ويمكن تغيير هذه القيمة لاحقاً أثناء تنفيذ البرنامج. المتغيرات تساعدك على إعادة استخدام القيم وتسهيل كتابة الكود.

كيفية إنشاء المتغيرات وتعيين القيم

لإنشاء متغير في Python، فقط اكتب اسم المتغير ثم علامة = ثم القيمة التي تريد تخزينها. لا تحتاج لتعريف نوع المتغير مسبقاً، فبايثون تحدد النوع تلقائياً. 

x = 10           # عدد صحيح
name = "محمد"    # نص (سلسلة محارف)
is_active = True # قيمة منطقية (صح أو خطأ)
pi = 3.14        # عدد عشري

تغيير قيمة المتغير

يمكنك تغيير قيمة المتغير في أي وقت: 

x = 5
x = 20  # الآن x تساوي 20 وليس 5

تسمية المتغيرات (الاسماء الصالحة)

  • يجب أن يبدأ اسم المتغير بحرف أو _ (ولا يبدأ برقم).
  • يمكن أن يحتوي على أرقام أو أحرف أو _ فقط.
  • لا يسمح بالمسافات أو الرموز الخاصة مثل ! أو @.
  • حساس لحالة الأحرف: age و Age متغيران مختلفان.
  • يفضل أن يكون الاسم معبراً عن محتوى المتغير.
_counter = 1
userName = "Ali"
age2 = 25
# أمثلة غير صحيحة:
# 2age = 10
# user-name = "Sara"

طباعة المتغيرات

استخدم دالة print() لعرض قيمة المتغير: 

print(x)
print(name)
print(is_active)

تعيين عدة متغيرات في سطر واحد

a, b, c = 1, 2, 3
print(a, b, c)  # الناتج: 1 2 3

إسناد نفس القيمة لعدة متغيرات

x = y = z = 0
print(x, y, z)  # الناتج: 0 0 0

نوع المتغير (type)

يمكنك معرفة نوع المتغير باستخدام الدالة type(): 

print(type(x))      # 
print(type(name))   # 
print(type(is_active)) #

ملاحظات هامة

  • لا تستخدم الكلمات المحجوزة في بايثون كأسماء متغيرات (مثل if، for، class).
  • يفضل كتابة أسماء المتغيرات باللغة الإنجليزية، لكن بايثون تدعم العربية أيضاً.
  • استخدم أسماء واضحة وسهلة الفهم.

بعد تعلم المتغيرات، انتقل إلى درس أنواع البيانات.
